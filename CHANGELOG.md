# 更新日志

## v0.3.4 - 2026-06-15

### 新增

- **电源直供电开关**（第五页）：插电时直接外部供电、不经电池充放，减少长期插电的电池损耗。走厂商 `zwrt_bsp.charger`（`set {direct_power_supply_mode:enable/disable}`，状态读 `list`）。令牌 `DPSCLASS/DPSSTATE`，动作 `act:dps`。
- **锁屏改版**：锁定后先显示**第一页实时预览**（数据刷新但不可操作）+ 屏幕中下方一个**原生锁图标**（随主题黑/白）；任意方向**滑动**才进 PIN 键盘。
- **电源菜单二次确认**：长按电源的关机/重启改为**点一次→按钮变「再按一次…」+白色描边（点击反馈）→再点执行**；执行时**先熄屏再发命令**（关机/重启有延迟，先黑屏给即时反馈）；4 秒无第二次点自动复原。
- **亮度持久化**：屏幕亮度写入 `/data/u60pro/devui.conf`，开机恢复、重装不丢。

### 修复

- **熄屏仍响应触摸（安全）**：触摸锁存队列在熄屏时照常入队，熄屏期间的点击会在亮屏后**回放**——曾导致熄屏时盲点也能把 PIN 输进去。现熄屏**完全忽略触摸并清队列**，亮屏首帧丢弃唤醒触摸。
- **锁频二级弹窗频段不清晰**：芯片颜色规则错挂在 `body.dark/.light`（弹窗 body 是 `mo`，从未命中）导致未选中项落到默认蓝字。改挂 `.modal.dark/.light`：未选中=边框轮廓+黑/白字，选中=蓝底白字。
- **已连接设备列表**：不再折叠（全部展开）；按**当前在线**过滤（WiFi 关联 ∪ ARP 在线），去掉 DHCP 租约残留的离线设备，且不误删 LAN/USB 接入设备。

### 移除

- **双击点亮**：连同熄屏触摸一并移除（熄屏不再检测触摸，唤醒只用电源键）。

## v0.3.3 - 2026-06-14

### 新增

- **锁屏（4 位 PIN）**：第五页设置卡新增「锁屏」开关。
  - 开关从关→开时进入**二级 PIN 键盘**设置 4 位密码（带「删除」键、无确认键，输完第 4 位自动保存并开启）；开→关直接清除密码、关闭锁屏。设置中可点「取消」放弃。
  - 开启后**屏幕超时或按电源键即自动锁屏**；开机若已设密码也直接进入解锁键盘。
  - 解锁键盘输满 4 位**自动校验**：正确进入界面，错误在**屏幕中间弹出「密码错误」**并清空重输。锁定时电源键长按不再弹电源菜单。
  - PIN 存于 `/data/ui/.lockpin`（持久、不被推送 `*.html` 覆盖）。忘记密码可 `adb shell rm /data/ui/.lockpin` 解除。
  - 新增界面文件 `lockscreen.html`（特殊页，不计入翻页）；令牌 `LOCKTITLE/PINDOTS/LOCKMSG/LOCKAUX`，动作 `locktoggle/pin:<0-9>/pin:del/lockcancel`。
- **真·圆角渲染**：之前自写的 litehtml 容器 `draw_solid_fill`/`draw_borders` 直接画矩形、忽略 `border-radius`，所以整个 UI 的圆角在设备上从未真正出现。新增 `fill_rounded`（圆角填充）与 `stroke_rounded`（等宽圆角描边），卡片/开关/按钮/电池外框等终于是真圆角。另加 `html_view_fill_poly`（扫描线多边形填充）。
- **充电闪电图标**：去掉原来的白色流光动画，改为充电时在电池上**原生绘制闪电**（随主题黑/白、贯穿电池上下、不超出状态栏）；电池压扁一点。
- **第二页 WiFi**：新增 **2.4G / 5G 频段开关**与 **WiFi 节能模式（PSM）开关**。频段用纯 `ifconfig wlanN up/down` 控制、读 `operstate`（本固件 uci/`wifi reload` 无效）；PSM 用 `iw set power_save` + hotplug 脚本持久化。令牌 `WIFI24*/WIFI5*/PSM*`，动作 `wifi24/wifi5/psm`。
- **设置持久化**：主题 / 速率单位 / 双击点亮 / 自动息屏 存到 `/data/u60pro/devui.conf`，开机加载、改动即写，重装不丢。
- **DHCP 地址池实时化**：改为 UI 直接由 uci（`network.lan.ipaddr` + `dhcp.lan.start/limit`）算出地址池范围，跟随实际网段。

### 修复

- **快速点击被吞**：触摸层新增点击边沿锁存 + 队列（`touch_input_take_tap`），一次完整按下→抬起若落在慢渲染期间也不再丢，锁屏键盘连点不掉键。
- **翻页时状态栏割裂**：横滑翻页时**钉住顶部状态栏**（只滑下方内容），闪电随之对齐、不再先消失再出现。
- **第四页网速单位写死**：右上角 ↓/↑ 数值原本恒为字节率（MB/s）、无视「速率单位」设置，现已跟随（Mbps/MB/s）。
- **第五页电池电流标正负**：充电 `+`、放电 `-`（与「充电/放电」标签一致）。

### 调整

- 设置页开关（`.sw`）整体放大并与文字**垂直居中**对齐（行改 table 布局）。

## v0.3.2 - 2026-06-14

### 新增

- **熄屏/亮屏淡入淡出**：熄屏时背光从当前亮度平滑降到 1 再灭、亮屏从 0 渐亮到熄屏前亮度；时长按亮度比例缩放（满亮度各约 0.125s）。淡变只改实时背光，**不影响第五页设置的亮度**（`backlight_get()` 返回记忆的用户亮度）。
- 亮屏前在背光为 0 时先把命令模式面板"热身"几帧，避开退出 idle 的瞬态闪烁（中高亮度已无闪）；并加最低档软启动以减轻低亮度过冲。

### 说明

- 淡变用真实时钟控时（写背光 sysfs 较慢，按步数计时会超时）。
- 极低亮度/偶发熄屏的轻微闪是背光升压电路冷启动/关断的硬件瞬态，纯软件难以根除。

### 新增

- **第五页系统卡**新增充电器/电池电压电流（充电器一行、电池一行，各两列，位于内存上方）。
- **第四页新增电池图表**：充电功率折线 + 电池温度折线，右上角显示温度与功率数值；**未充电时显示放电功率**（电池 V×I）。
- 后端 `u60-datad` 新增 `battery.chg_uv/chg_ua/bat_uv/bat_ua`（充电器/电池电压µV、电流µA，读自 `power_supply` sysfs）。

### 调整

- **自动息屏**改为与选网方式一致的**分段控件**（关/30秒/1/2/5/10分，蓝色高亮跟手滑动 + 单击）。
- 分段控件单元格显式等宽（litehtml 不自动平分 `table-layout:fixed`）。
- 删除可滚动页右侧的滚动条指示（滚动功能保留）。

## v0.2.0 - 2026-06-13

这一版是一次**架构级改动**：界面从内置的 LVGL 代码改为**运行时渲染 HTML/CSS**。程序变成固定的"框架"，真正的界面是 `/data/ui` 目录里的 HTML 文件，用户改界面不用再重新编译。详见新增的 [docs/UI-GUIDE.md](docs/UI-GUIDE.md)。

### 架构

- **改用 litehtml + FreeType 渲染 HTML/CSS** 到 RGB565 framebuffer，全静态链接 aarch64 musl 单文件。
- 界面与程序解耦：`/data/ui/NN-名字.html` 每个文件一页，`style.css` 共享样式，`menu.html` 为电源菜单。
- HTML 里的 `{{令牌}}` 在渲染前由程序替换为实时数据；`href="act:xxx"` 为交互动作。每次刷新重读文件，改完即时生效。
- 后端从 `zwrt-datad` 切换为 `u60-datad`，快照路径统一为 `/tmp/u60-datad/state.json`。

### 新增

- **信号页**：按载波分卡片显示 频段·频宽 / PCI / RSRP / SINR，按质量上色；未激活副载波（RSRP 为 -140 哨兵值）置灰并标"未激活"。
- **状态栏**：时间 · 实时网速(`↑上 ↓下`，单位可切) · 网络代际文字(5GA/5G+/5G/4G/LTE/3G) · 格数信号强度 · 电池(带充电流光) · 电量。
- **WiFi 页**：SSID、密码（默认打码，点按才明文）、加密方式、设备数。
- **系统/设置页**：速率流量、PCI、ARFCN（默认隐藏）、型号/运行/温度/内存，以及 ADB 调试 / 速率单位 / 界面主题三个开关。
- **网络代际徽章**：按 MCC/MNC + 载波聚合判定 5GA / 5G+ / 5G / 4G / LTE / 3G。
- **运营商中文名**：中国移动 / 联通 / 电信 / 广电。
- **QCI / AMBR**：从后端 QoS 读取并显示（AMBR 统一 Mbps）。
- **跟手翻页动画**、**充电流光动画**（宿主驱动帧，绕过 litehtml 无 CSS 动画的限制）。
- **底部圆点**翻页指示。
- 文档：新增《自定义界面教程》[docs/UI-GUIDE.md](docs/UI-GUIDE.md)；DEVELOPMENT.md 全面重写为 HTML 架构。

### 修复

- 纯 NR-SA 下误显示 LTE 载波（modem 残留 lte_rsrp）。
- 后端补回/新增 `nrca`/`lteca`/`usb_mode`/`wlan` 字段（缺 `wlan` 会导致 WiFi 页读空）。
- 状态栏制式徽章背景色因裸 class 名泄漏到纯文字 span（紫色框），改为带 `.gen` 前缀。
- 状态栏文字与 inline-block 元素对齐错位；电池正极脱节。

### 安全

- WiFi 密码、ARFCN 默认打码，需显式点按才明文显示。
