# 更新日志

## v0.3.9 - 2026-06-19

### 修复

- **电源菜单跟随长页面滚动偏移**：设置页滑到底后长按电源，`menu.html` 会沿用当前页的 `g_scroll`，导致电源菜单整体被卷走、只剩「取消」露在顶部。现将 `menu.html` / `lockscreen.html` 这类特殊全屏页强制以 `scroll=0` 渲染，关闭菜单后普通页面仍保留原滚动位置。
- **关机插电应进入原厂充电动画**：原自启逻辑在 `/etc/rc.local` 中无条件停掉 `zte_topsw_devui` 并启动 DevUI，导致关机插电的离线充电动画被 DevUI 抢走。现自启先检查 `/proc/cmdline`，只有明确包含 `silent_boot.mode=nonsilent` 的正常开机才接管屏幕；离线充电启动保持原厂 `ANIM_MODE_POWER_OFF_CHARGER`。
- **管理插件更新 UI 后页面重复**：插件更新 `ui.tar.gz` 时会先解压并确认新包里有页面，再清空 `/data/ui` 顶层旧 `*.html` / `*.css`，最后复制新模板并核对数量，避免旧 4 页 + 新 4 页叠成 8 页；同时新增「重装UI」按钮，远端版本没变时也能手动重铺模板自愈。
- **短信状态栏图标错位/随滚动漂移**：未读短信信封图标改为按状态栏固定坐标绘制在时间右侧，不再依赖页面里的占位元素，长页面滚动时不会变位置。
- **短信页只能显示 6 条**：短信数据结构和列表缓冲扩大到 32 条，短信页内容超过一屏后可向下滚动查看。
- **页面滑动延迟和快速滚动卡顿**：屏幕亮着时主循环触摸轮询间隔从约 30ms 降到约 8ms，拖动起始阈值从 14px 降到 10px；竖向滚动帧只刷新状态栏下方内容区，不再每帧重画固定状态栏，减轻快速上下拖动时的卡顿。
- **亮屏淡入动画偏慢**：亮屏时背光淡入时长缩短约一半；息屏淡出时长保持不变。

### 验证

- 已实测：关机状态插电进入原厂充电画面；正常开机进入 DevUI。
- 已实测：流畅性优化版已部署到设备，进程正常接管 DRM/触摸，帧缓冲健康检查不是黑屏。
- 已实测：亮屏淡入减半版已部署到设备，`u60pro-devui` 正常启动。

## v0.3.8 - 2026-06-19

### 新增

- **USB-C 供电方向开关**：设置页新增「USB供电方向」，可在给 U60 充电和 U60 反向供电之间切换；网络共享开启时会走更稳的 Type-C/USB 顺序，避免开关状态来回弹。
- **USB 网络共享开关**：设置页新增「USB网络共享」，移除原设置页 ADB 开关。默认使用系统 `9057` composition（RNDIS : ECM），电脑侧走 RNDIS；手机侧若 `9057` 无 carrier，会自动降到纯 ECM `90B1`。
- **状态栏电量百分比开关**：默认开启，可在设置页隐藏/显示电池框内百分比。

### 变更

- **状态栏重绘**：时间靠左，网速靠近信号区；信号条、5G/5GA/5G+ 标识、电池框和电量百分比改为原生绘制并重新对齐。
- **状态栏常驻**：页面纵向滚动和横向切页时，状态栏保持钉在顶部，不再随内容消失又回来。
- **电池百分比可读性**：电量填充按百分比绘制，充电时填充为绿色；百分比文字根据背景实时选择高对比颜色。

### 修复

- **电脑/手机 USB 网络自适应**：修复从手机 ECM 切回电脑时粘在 `90B1/ECM` 的问题；watchdog 会检测 `ecm0/rndis0 carrier`，必要时清掉残留 source 标记并回到 `9057/RNDIS`。
- **手机反向供电 + ECM 共享**：按实测顺序先建立 ECM carrier，再切 `source/device` 和 powerbank，保持 U60 给手机供电的同时向手机共享网络。
- **设置页语言和排版回归**：恢复中国运营商中文显示相关文档记录，修正设置页新增开关后的布局说明。

### 说明

## v0.3.6 - 2026-06-16

### 变更

- **电源菜单改版（长按电源键弹出）**：原来上半屏三个长方形开关，改为**三个竖排大圆形按键**，圆盘和图标全部原生绘制（litehtml 画不了这些符号）——关机=电源符号（红）、重启=圆箭头（橙）、取消=X（灰），每个圆下方标文字。二次确认保留：第一次点亮白圈光环、标签变「再按一次」。

### 修复

- **WiFi 节能模式开关开了不生效/会自己关闭**：`power_save` 需要靠 `/etc/hotplug.d/iface/99-disable-powersave` 这类 hotplug 脚本在每次 `ifup` 时重新应用，否则切频段/重连/重启后会被弹回默认（关闭）。原本该脚本要手动安装、且「节能 ON」只是一次性 `iw set on`，导致没装脚本或开启节能的用户都不能保持。现改为**开关自己把脚本 echo 到 `/etc/hotplug.d/iface/99-disable-powersave`**（按所选模式在 ifup 时重新应用 on/off），功能自包含、无需另行安装；同时清掉旧版遗留的 `psm` 文件。

## v0.3.5 - 2026-06-15

### 新增

- **短信查看（只读）**：新增第二页「短信」，按时间倒序列出最近 6 条。列表为**折叠卡片**——每条只显示号码 / 时间 + 一行正文预览；**点卡片弹出二级页面**看完整内容，点「关闭」或弹窗外区域返回。只读，不支持发送。后端走厂商 `zwrt_wms`：未读数读 `zwrt_wms_get_wms_capacity`，列表读 `zte_libwms_get_sms_data`；正文是 UTF-16BE 十六进制，后端解码成 UTF-8（含 emoji 代理对），日期格式化成 `MM-DD HH:MM`。令牌 `SMSLIST`，未读条目左侧红点 + 蓝色号码高亮。
- **点开短信自动标为已读**：进入二级页面时，若该条未读则调用 `zwrt_wms_modify_tag {id, tag:0}` 标为已读；未读数变化后后端立即刷新列表，红点与状态栏图标随之消失。
- **状态栏未读短信图标**：检测到未读短信时，时间右侧**原生绘制一个信封图标**（蓝色信封 + 白色翻盖，非字体字形）。状态栏置顶、翻页时也常驻。**锁屏状态栏同样显示**（锁屏复用 `STATUSBAR`）。

### 变更

- **锁频二级弹窗频段芯片一行四个**：原 3 列改 4 列。弹窗内宽约 264px，芯片 56→52px、外边距 4→3px、字号 15→14px，确保四个并排不换行。
- **页面重新编号**：原 02-wifi / 03-lock / 04-charts / 05-system 顺延为 03 / 04 / 05 / 06，短信页插入为 02。

### 注意

- 短信正文可能含任意字符，UI 端对 `&<>` 做了 HTML 转义后再渲染。

## v0.3.3 - 2026-06-14

### 新增

- **锁屏（4 位 PIN）**：第五页设置卡新增「锁屏」开关。
  - 开关从关→开时进入**二级 PIN 键盘**设置 4 位密码（带「删除」键、无确认键，输完第 4 位自动保存并开启）；开→关直接清除密码、关闭锁屏。设置中可点「取消」放弃。
  - 开启后**屏幕超时或按电源键即自动锁屏**；开机若已设密码也直接进入解锁键盘。
  - 解锁键盘输满 4 位**自动校验**：正确进入界面，错误在**屏幕中间弹出「密码错误」**并清空重输。锁定时电源键长按不再弹电源菜单。
  - PIN 存于 `/data/ui/.lockpin`（持久、不被推送 `*.html` 覆盖）。忘记密码可 `adb shell rm /data/ui/.lockpin` 解除。
  - 新增界面文件 `lockscreen.html`（特殊页，不计入翻页）；令牌 `LOCKTITLE/PINDOTS/LOCKMSG/LOCKAUX`，动作 `locktoggle/pin:<0-9>/pin:del/lockcancel`。
- **真·圆角渲染**：之前自写的 litehtml 容器 `draw_solid_fill`/`draw_borders` 直接画矩形、忽略 `border-radius`，所以整个 UI 的圆角在设备上从未真正出现。新增 `fill_rounded`（圆角填充）与 `stroke_rounded`（等宽圆角描边），卡片/开关/按钮/电池外框等终于是真圆角。另加 `html_view_fill_poly`（扫描线多边形填充）。
- **充电闪电图标**：去掉原来的白色流光动画，改为充电时在电池上**原生绘制闪电**（随主题黑/白、贯穿电池上下、不超出状态栏）；电池压扁一点。
- **第二页 WiFi**：新增 **2.4G / 5G 频段开关**与 **WiFi 节能模式（PSM）开关**。频段用纯 `ifconfig wlanN up/down` 控制、读 `operstate`（本固件 uci/`wifi reload` 无效）；PSM 用 `iw set power_save` + hotplug 脚本持久化。令牌 `WIFI24*/WIFI5*/PSM*`，动作 `wifi24/wifi5/psm`。
- **设置持久化**：主题 / 速率单位 / 双击点亮 / 自动息屏 存到 `/data/u60pro/devui.conf`，开机加载、改动即写，重装不丢。
- **DHCP 地址池实时化**：改为 UI 直接由 uci（`network.lan.ipaddr` + `dhcp.lan.start/limit`）算出地址池范围，跟随实际网段。

### 修复

- **快速点击被吞**：触摸层新增点击边沿锁存 + 队列（`touch_input_take_tap`），一次完整按下→抬起若落在慢渲染期间也不再丢，锁屏键盘连点不掉键。
- **翻页时状态栏割裂**：横滑翻页时**钉住顶部状态栏**（只滑下方内容），闪电随之对齐、不再先消失再出现。
- **第四页网速单位写死**：右上角 ↓/↑ 数值原本恒为字节率（MB/s）、无视「速率单位」设置，现已跟随（Mbps/MB/s）。
- **第五页电池电流标正负**：充电 `+`、放电 `-`（与「充电/放电」标签一致）。

### 调整

- 设置页开关（`.sw`）整体放大并与文字**垂直居中**对齐（行改 table 布局）。

## v0.2.0 - 2026-06-13

这一版是一次**架构级改动**：界面从内置的 LVGL 代码改为**运行时渲染 HTML/CSS**。程序变成固定的"框架"，真正的界面是 `/data/ui` 目录里的 HTML 文件，用户改界面不用再重新编译。详见新增的 [docs/UI-GUIDE.md](docs/UI-GUIDE.md)。

### 架构

- **改用 litehtml + FreeType 渲染 HTML/CSS** 到 RGB565 framebuffer，全静态链接 aarch64 musl 单文件。
- 界面与程序解耦：`/data/ui/NN-名字.html` 每个文件一页，`style.css` 共享样式，`menu.html` 为电源菜单。
- HTML 里的 `{{令牌}}` 在渲染前由程序替换为实时数据；`href="act:xxx"` 为交互动作。每次刷新重读文件，改完即时生效。
- 后端从 `zwrt-datad` 切换为 `u60-datad`，快照路径统一为 `/tmp/u60-datad/state.json`。

### 新增

- **信号页**：按载波分卡片显示 频段·频宽 / PCI / RSRP / SINR，按质量上色；未激活副载波（RSRP 为 -140 哨兵值）置灰并标"未激活"。
- **状态栏**：时间 · 实时网速(`↑上 ↓下`，单位可切) · 网络代际文字(5GA/5G+/5G/4G/LTE/3G) · 格数信号强度 · 电池(带充电流光) · 电量。
- **WiFi 页**：SSID、密码（默认打码，点按才明文）、加密方式、设备数。
- **系统/设置页**：速率流量、PCI、ARFCN（默认隐藏）、型号/运行/温度/内存，以及 ADB 调试 / 速率单位 / 界面主题三个开关。
- **网络代际徽章**：按 MCC/MNC + 载波聚合判定 5GA / 5G+ / 5G / 4G / LTE / 3G。
- **运营商中文名**：中国移动 / 联通 / 电信 / 广电。
- **QCI / AMBR**：从后端 QoS 读取并显示（AMBR 统一 Mbps）。
- **跟手翻页动画**、**充电流光动画**（宿主驱动帧，绕过 litehtml 无 CSS 动画的限制）。
- **底部圆点**翻页指示。
